# Task 2 – Credit Risk Prediction

Objective:
Predict whether a loan applicant is likely to default using machine learning.

Approach:

Cleaned missing data and handled categorical variables.
Explored features like loan amount, income, and education with visualizations.
Trained a Decision Tree / Logistic Regression model and evaluated using accuracy and confusion matrix.

Results & Insights:

Income, loan amount, and education are key factors influencing defaults.
Most cases were predicted correctly, with a few borderline misclassifications.
Learned how data cleaning, visualization, and model evaluation work together for prediction.
